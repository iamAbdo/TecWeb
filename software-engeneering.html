Software Engineering

Software engineering is a discipline that combines principles of computer science, engineering, and project management to design, develop, test, and maintain software systems. It is a systematic approach to creating software that meets specific requirements, is reliable, and can be maintained over time.

Certificats

Certified Software Development Professional (CSDP)

Description: This certification offered by the IEEE Computer Society, focuses on software engineering principles, lifecycle processes, and best practices.

Certified Software Quality Analyst (CSQA)

Description: Focuses on quality assurance principles and methods in software engineering.

Microsoft Certified: Azure Developer Associate

Description: This certification demonstrates skills in developing cloud applications on Microsoft Azure.

Google Professional Machine Learning Engineer

Description: This certificate Validates expertise in designing and building machine learning models on Google Cloud.

Project Management Professional (PMP)

Description:This certificat validates project management skills, including software projects.

Certified Kubernetes Application Developer (CKAD)

Description: This certificat focuses on designing and deploying containerized applications using Kubernetes.
